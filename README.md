# Gerador de Senhas Seguras

Aplicação web desenvolvida para gerar senhas fortes e personalizadas de forma rápida e segura.

# Funcionalidades

✔ Geração instantânea de senha  
✔ Definição de tamanho da senha  
✔ Opção de incluir:
- Letras maiúsculas
- Letras minúsculas
- Números
- Caracteres especiais  

✔ Botão para copiar senha  
✔ Interface simples e intuitiva  
✔ Layout responsivo  

---

#Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript Vanilla  

---

# Lógica de Funcionamento

O sistema:

1. Cria um conjunto de caracteres baseado nas opções selecionadas
2. Gera uma senha aleatória utilizando `Math.random()`
3. Exibe a senha gerada na tela
4. Permite copiar automaticamente para a área de transferência
